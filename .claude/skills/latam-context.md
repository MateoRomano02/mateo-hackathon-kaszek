# Skill: LATAM Context
Cuando trabajés en este proyecto tené en cuenta:
- Usuarios objetivo: Argentina/LATAM
- Idioma de la UI: español neutro (no lunfardo en la UI, sí en comentarios)
- APIs relevantes del ecosistema: MercadoLibre, MercadoPago, Nuvemshop, AFIP, Pomelo, Tapi
- Formato fecha: DD/MM/YYYY
- Formato número: 1.000,00 (punto miles, coma decimales)
- Contexto económico: inflación, dólar, facturación electrónica
- El usuario técnico típico: founder o CTO de startup LATAM, inglés técnico ok
