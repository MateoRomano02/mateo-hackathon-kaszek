## Criterios de evaluación (de hackathons Anthropic anteriores)
1. **Implementación técnica** — tool use real, flujos agénticos, NO solo chatbot wrapper
2. **Problema e impacto** — usuario claro, pain point real, camino creíble a adopción
3. **Creatividad** — ángulo fresco, uso inesperado de Claude
4. **Demo funcional** — prototipo que funciona > documentación perfecta

## Lo que gana en hackathons de Anthropic
- Problema REAL y ESPECÍFICO (no genérico)
- Claude usado de forma sofisticada (tool use + agents)
- Demo que se puede ver y tocar
- Story de impacto clara en 2 minutos
- Ganadores anteriores: cardiólogo, abogado, músico — no necesariamente devs
